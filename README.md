# Sea of Thieves map locations
## Init
Install packages:
```bash
npm install
```

Login to firebase (to deploy):
```bash
firebase login
```

## Scripts
- Build:
```bash
npm run build
```

- Build only Json `./data/locations.json`:
```bash
npm run build-json
```

- Build only Json `./public/index.html`:
```bash
npm run build-html
```

- Publish:
```bash
npm run publish
```
